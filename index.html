Politique de confidentialité – ImmoScan AI

Dernière mise à jour : avril 2026

Données collectées

ImmoScan AI ne collecte, ne stocke et ne transmet aucune donnée personnelle à ses propres serveurs. L'extension ne dispose d'aucun serveur tiers.

Clé API Anthropic

La clé API Anthropic que vous saisissez est stockée uniquement dans le stockage local de votre navigateur (chrome.storage.local). Elle ne quitte jamais votre machine, sauf pour être transmise directement à l'API Anthropic lors d'une analyse.

Données envoyées à Anthropic

Lors d'une analyse, le contenu de l'annonce immobilière (texte et éventuellement photos) est transmis à l'API Claude d'Anthropic afin de générer l'analyse. Ces échanges sont régis par la politique de confidentialité d'Anthropic.

Cache local

Les résultats d'analyse sont mis en cache localement dans votre navigateur pour éviter les appels API redondants. Ces données restent sur votre machine et ne sont jamais partagées.

Contact

Pour toute question, ouvrez une issue sur le dépôt GitHub de l'extension.
